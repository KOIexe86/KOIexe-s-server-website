# KOIexe's server website

## Overview
This is the website for KOIexe's server. It is used to showcase my server.

## Folder Structure
```
index.html           # Main website entry point
css/                 # Stylesheets
	style.css
fonts/               # Custom fonts
images/              # Images and icons
	applications/      # Application-specific images
js/                  # JavaScript files
	main.js
NodeJsProxy/         # Node.js proxy scripts
	NodeJSproxy.js
```
