The story

KOIexe's Server was built in summer of 2025 as a personal project to learn about server administration, virtualization and to have usefull services. The server was built using my previous computer and is running Proxmox VE, a popular open-source virtualization platform.

At the start, used for mincraft server hosting for me and my friends, it became a multipurpose server a bit later. I started by adding storage capability then, more variaty in my services. Now, it is used by me and my friends for file sharing, storage, website hosting and, still to this day, minecraft hosting.
